# Описание проекта: Виджет банковских операций клиента
## О проекте
Проект содержит набор функций для обработки и отображения банковских операций клиента. 
Реализована возможность маскировки информации, 
фильтрации и сортировки операций

## Установка:
### Требования:
- Python 3.13

## Функции
### Маскировка данных
### get_mask_card_number
Частично маскирует номер банковской карты
#### Параметры:
- card_number(str): номер карты
#### Возвращаемое значение:
- str: замаскированный номер карты
#### Пример:
7000 79 ** **** 6361

### get_mask_account
Маскирует номер банковского счета
#### Параметры:
- account_number(str): номер счета
#### Возвращаемое значение:
- str: замаскированный номер счета
#### Пример:
Счет **4305

### mask_account_card
Маскирует одновременно номер счета и карты
#### Параметры:
- account_number(str): номер счета
- card_number(str): номер карты
#### Возвращаемое значение:
str: (замаскированный номер счета или замаскированный номер карты)
#### Пример:
Счет **4305

### Работа с операциями
### filter_by_state
Фильтрует список операций по значению ключа
#### Параметры:
- list_of_dict(list): список словарей
- state(str): значение ключа для фильтрации (по умолчанию - ‘EXECUTED’)
#### Возвращаемое значение:
- list: отфильтрованный список операций
#### Пример:
#### Выход функции со статусом по умолчанию 'EXECUTED'
[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}]

### sort_by_date
Сортирует список операций по дате
#### Параметры:
- operation_list(list): список словарей с операциями
- reverse_sort(bool): порядок сортировки - по убыванию
#### Возвращаемое значение:
- list: отсортированный список операций
#### Пример:
#### Выход функции (сортировка по убыванию)
[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}]
